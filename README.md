# ytup

Official downloads for **ytup** — a Windows desktop application for downloading
media with yt-dlp.

## Download

Download the current installer from the
[latest release](../../releases/latest).

## Requirements

- Windows 10 version 22H2 or newer, or Windows 11
- 64-bit Windows

## Installation

1. Download `ytup-0.1.0-windows-x64.exe` from the latest release.
2. Run the installer.
3. Start **ytup** from the Start menu.

The installer uses a per-user installation and normally does not require
administrator permissions.

## Important

Only download media that you are permitted to save and use. Availability,
format selection, and download success depend on the source website.

ytup is not affiliated with YouTube, Facebook, Meta, Google, or any other
supported website.
