# ytup

Official downloads for **ytup** — a Windows desktop app for downloading media with yt-dlp.

## Download

Download the newest installer from the [latest release](../../releases/latest).

## Features

* Download video as MP4
* Extract audio as MP3
* Download playlists with a selectable item limit
* Download multiple URLs in sequence
* View download history
* Works with websites supported by yt-dlp

## Requirements

* Windows 10 or newer
* 64-bit Windows

## Installation

1. Download `ytup-0.1.0-windows-x64.exe` from the latest release.
2. Run the installer.
3. Start **ytup** from the Start menu.

The installer uses a per-user installation and normally does not require administrator permissions.

## Important

Only download media that you are permitted to save and use. Availability and supported formats depend on the source website.

ytup is not affiliated with YouTube, Facebook, or any other supported website.
