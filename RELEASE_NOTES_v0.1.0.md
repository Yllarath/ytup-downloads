# ytup 0.1.0

## First public Windows release

### Included features

- MP4 video downloads
- MP3 audio extraction
- Playlist downloads with a configurable limit
- Multiple URLs processed sequentially
- Download cancellation
- Download history
- Bundled FFmpeg, FFprobe, and Deno runtime
- Per-user installer with Start-menu shortcut and optional desktop shortcut

### System requirements

- Windows 10 version 22H2 or newer, or Windows 11
- 64-bit Windows
